## Bibliotēkas mājaslapa

Šis projekts tika izstrādāts kā mans WEB noslēguma darbs. Tā mērķis bija izveidot sistēmu lauku bibliotēkām, kas ļautu lietotājiem ērti pārlūkot grāmatu klāstu un iepriekš rezervēt sev interesējošās grāmatas.

[Skatīt projektu tiešsaistē](https://kristovskis.lv/4pt/silders/Biblioteka)

### Projekta apraksts

Bibliotēkas mājaslapa nodrošina lietotājiem iespēju meklēt, rezervēt un apskatīt grāmatas tiešsaistē.
Lietotāji var redzēt, kuras grāmatas ir pieejamas, un veikt rezervācijas savā kontā.
Visi dati — grāmatas, lietotāji un rezervācijas, tiek glabāti datubāzē un ir pārvaldāmi administrācijas panelī.

---

### Galvenā funkcionalitāte

#### Lietotāja iespējas

* Pārlūkot un meklēt grāmatas.
* Rezervēt grāmatas tiešsaistē.
* Izveidot lietotāja kontu un pieteikties sistēmā.

#### Administratora iespējas

* Pievienot, rediģēt un dzēst datus par grāmatām, lietotājiem un rezervācijām.
* Pārraudzīt lietotāju aktivitāti un rezervācijas.
* Apskatīt statistiku, piemēram:

  * Kopējo rezervēto grāmatu skaitu
  * Rezervācijas pēdējo 7 dienu laikā
  * Kopējo grāmatu un lasītāju skaitu

---

### Izmantotās tehnoloģijas

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Datubāze:** MySQL
* **Maksājumu sistēma:** Stripe API (testa režīmā)
* **Administrācijas panelis:** PHP + MySQL

## IT ir Spēks

Šī mājaslapa tika izstrādāta kā 3. kursa noslēguma projekts. Tā piedāvā lietotājiem iespēju atrast un pieteikties IT nozares vakancēm, kā arī sekot līdzi jaunākajām aktualitātēm tehnoloģiju pasaulē.

[Skatīt projektu tiešsaistē](https://kristovskis.lv/4pt/k12/grupa-IlIlIl/)

### Projekta apraksts

IT ir Spēks ir platforma, kas apvieno darba devējus un interesentus IT jomā.
Lietotāji var pārlūkot aktuālās vakances, pieteikties tām un lasīt jaunākās ziņas par tehnoloģiju tendencēm.
Projekts ietver arī administrācijas sadaļu ar plašām datu pārvaldības iespējām.

---

### Galvenā funkcionalitāte

#### Lietotāja iespējas

* Pārlūkot un pieteikties IT vakancēm.
* Apskatīt jaunākās IT ziņas un aktualitātes.
* Reģistrēties un autorizēties sistēmā.

#### Administratora iespējas

* Pārvaldīt lietotājus, vakances un ziņas.
* Apskatīt statistiku par aktivitātēm un lietotājiem.
* Pievienot, rediģēt un dzēst ierakstus datubāzē.

---

### Izmantotās tehnoloģijas

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Datubāze:** MySQL
